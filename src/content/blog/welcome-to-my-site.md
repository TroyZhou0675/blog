## 为什么搭建这个网站

过去一直想拥有一个个人站来分享和交流知识，但没有 Web 开发技术，计划一直搁置。

后来发现了 Claude Code，现在终于把这个想法变成了现实。我希望通过这个网站分享平时学到的新技术、读书笔记，以及核磁共振的学习过程。

## 技术选型

- **Astro 4** — 静态生成，无 JS 运行时，性能极好，适合内容为主的书呆站
- **Obsidian → Markdown → Content Collections** — 笔记直接在 Obsidian 编辑，自动同步到网站
- **KaTeX** — 数学公式渲染，MRI 和信号处理的笔记里有很多公式
- **GitHub Pages + CI/CD** — 推送到 main 分支，自动部署到 troyzhou.top
- **Gitalk** — 博客文章下方的评论区，基于 GitHub Discussion
- **Claude Code** — 服务器的搭建、站点的美化与日常维护，全靠后台自动进行

整个迁移流程已经跑通：Obsidian 写笔记 → 放 Interact/Markdown/ → AI 自动发布。
