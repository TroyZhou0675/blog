1. 串口标准化打印可以通过printf重定向完成
重新定义putc，链接器会优先采用你的定义，从而将原有定义覆盖
```
#include "stdio.h" // 确保包含头文件

int fputc(int ch, FILE *f)
{
    USART_SendData(USART1, (uint8_t)ch);
    while ((USART1->SR & USART_SR_TC) == 0); // 等待发送完成
    return ch;
}

```
2. 标准库的USART_Send_data()看上去一次发送两个字节，实际上高位是掩码，实际传输的只有低8位
```
void USART_SendData(USART_TypeDef* USARTx, uint16_t Data)
{
  /* Check the parameters */
  assert_param(IS_USART_ALL_PERIPH(USARTx));
  assert_param(IS_USART_DATA(Data)); 
    
  /* Transmit Data */
  USARTx->DR = (Data & (uint16_t)0x01FF);
}

```
可以进一步封装成：
```
void USART_SendByte(USART_TypeDef* USARTx, uint8_t ch)
{
    USARTx->DR = ch;
    while (!(USARTx->SR & USART_SR_TC)); // 等待发送完成
}

```
