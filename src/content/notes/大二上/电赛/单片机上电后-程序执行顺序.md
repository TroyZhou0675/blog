上电后执行startup.s的汇编文件
```
.syntax unified
.cpu cortex-m4
.thumb

.section .isr_vector
g_pfnVectors:
  .word _estack                // 栈顶指针
  .word Reset_Handler          // 复位中断处理函数
  .word NMI_Handler
  .word HardFault_Handler
  ...

Reset_Handler:
  LDR  R0, =_estack
  MOV  SP, R0
  BL   SystemInit
  BL   __main

```
1. 确定栈顶指针
2. 确定中断向量表
3. 执行reste_handler
在reset中断处理函数中会执行System_Init()函数
