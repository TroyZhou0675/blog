### **路原理训练题目：应用网孔分析法求解多回路电路**

**背景 (Background):**  
此题目用于训练AI助教模型。模型需要学习如何仅通过文字描述来构建电路的数学模型，并应用网孔分析法（或称回路电流法）进行求解。核心考察知识点包括：基尔霍夫电压定律 (KVL)、欧姆定律以及联立线性方程组的求解。

**电路描述 (Circuit Description):**  
请想象一个直流电路，它由两个直流电压源和三个电阻构成。电路中的节点被标记为 A, B, C 和一个公共的参考节点（或地线）D。

元件连接关系如下：

- **电源 V1:** 一个 30V 的理想电压源，其正极连接到节点 A，负极连接到参考节点 D。
    
- **电阻 R1:** 一个 5Ω 的电阻，连接在节点 A 和节点 B 之间。
    
- **电阻 R2:** 一个 10Ω 的电阻，连接在节点 B 和参考节点 D 之间。
    
- **电阻 R3:** 一个 20Ω 的电阻，连接在节点 B 和节点 C 之间。
    
- **电源 V2:** 一个 40V 的理想电压源，其正极连接到节点 C，负极连接到参考节点 D。
    

**问题 (Questions):**

1. **定义网孔电流:** 在电路中可以识别出两个独立的网孔（回路）。
    
    - 左侧网孔（Mesh 1）包含 V1、R1 和 R2。
        
    - 右侧网孔（Mesh 2）包含 R2、R3 和 V2。  
        请为这两个网孔分别定义一个顺时针方向的网孔电流，记为 i1 和 i2。
        
2. **列出KVL方程:** 对网孔1和网孔2分别应用基尔霍夫电压定律 (KVL)，写出关于 i1 和 i2 的两个线性方程。
    
3. **求解网孔电流:** 求解您在问题2中建立的方程组，计算出网孔电流 i1 和 i2 的精确值（可以使用分数表示）。
    
4. **计算节点电压:** 计算节点 B 相对于参考节点 D 的电压值，即 V_B。

### **参考答案与解题步骤 (Reference Answer and Solution Steps)**

这是为助教模型提供的标准答案和详细解析，模型应学习按照这个逻辑和步骤进行解释。

**第一步：识别网孔并定义网孔电流**

- 电路包含两个网孔。
    
- **网孔1 (左侧):** 我们定义顺时针方向的电流为 i1。该电流流过 V1, R1 和 R2。
    
- **网孔2 (右侧):** 我们定义顺时针方向的电流为 i2。该电流流过 R2, R3 和 V2。
    
- **注意:** 电阻 R2 是两个网孔的公共元件。流过 R2 的实际电流是 i1 和 i2 叠加的结果。
    

**第二步：对每个网孔应用KVL列写方程**  
根据KVL，在一个闭合回路中，所有电压的代数和为零。我们通常将电压源的“升压”视为正，电阻上的“压降”视为负。

- **对于网孔1:**
    
    - 从参考点D开始，顺时针方向：
        
    - 经过 30V 电源 V1 (从负到正)，电压上升: +30
        
    - 经过 5Ω 电阻 R1，压降为: -R1 * i1 = -5*i1
        
    - 经过 10Ω 电阻 R2，该支路上有两个电流作用。从上到下看，i1 向下流，i2 向上流。因此，净电流为 (i1 - i2) 向下。压降为: -R2 * (i1 - i2) = -10*(i1 - i2)
        
    - **KVL 方程 1:**  
        30 - 5*i1 - 10*(i1 - i2) = 0
        
    - **化简:**  
        30 - 5*i1 - 10*i1 + 10*i2 = 0  
        -15*i1 + 10*i2 = -30  
        两边同除以 -5，得： **3*i1 - 2*i2 = 6 (方程一)**
        
- **对于网孔2:**
    
    - 从节点B开始，顺时针方向：
        
    - 经过 20Ω 电阻 R3，压降为: -R3 * i2 = -20*i2
        
    - 经过 40V 电源 V2 (从正到负)，电压下降: -40
        
    - 经过 10Ω 电阻 R2，我们是逆着 i1 方向，顺着 i2 方向。从下到上看，i2 向上流，i1 向下流。净电流为 (i2 - i1) 向上。由于我们是向上走的，这是压降: -R2 * (i2 - i1)。更简单的方法是直接写 +10*(i1 - i2)。
        
    - **KVL 方程 2:**  
        -20*i2 - 40 + 10*(i1 - i2) = 0
        
    - **化简:**  
        10*i1 - 20*i2 - 10*i2 - 40 = 0  
        10*i1 - 30*i2 = 40  
        两边同除以 10，得： **i1 - 3*i2 = 4 (方程二)**
        

**第三步：求解线性方程组**  
现在我们有方程组：

1. 3*i1 - 2*i2 = 6
    
2. i1 - 3*i2 = 4
    

- **求解方法 (例如，代入消元法):**
    
    - 从方程二中，可得 i1 = 3*i2 + 4。
        
    - 将此表达式代入方程一：  
        3*(3*i2 + 4) - 2*i2 = 6
        
    - 展开并求解 i2：  
        9*i2 + 12 - 2*i2 = 6  
        7*i2 = -6  
        **i2 = -6/7 A**
        
    - 将 i2 的值代回 i1 的表达式：  
        i1 = 3*(-6/7) + 4 = -18/7 + 28/7  
        **i1 = 10/7 A**
        
- **结果解读:** i1 为正值，表示其真实方向与我们假设的顺时针方向一致。i2 为负值，表示其真实方向与我们假设的顺时针方向相反，实际上是逆时针流动的。
    

**第四步：计算节点电压 V_B**  
节点 B 的电压 V_B 是指 B 点与参考点 D 之间的电位差。这可以通过计算流过电阻 R2 的实际电流和 R2 的阻值来得到。

- **计算流过 R2 的实际电流 (I_R2):**  
    我们定义的 i1 向下流过 R2，i2 向上流过 R2。因此，从 B 到 D (向下) 的净电流为：  
    I_R2 = i1 - i2  
    I_R2 = (10/7) - (-6/7) = 10/7 + 6/7 = 16/7 A
    
- **计算 V_B:**  
    由于净电流 I_R2 是从 B 指向 D 的正值，说明 B 点的电位高于 D 点。根据欧姆定律：  
    V_B = I_R2 * R2  
    V_B = (16/7 A) * 10Ω  
    **V_B = 160/7 V (约等于 22.86V)**
