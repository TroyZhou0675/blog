# 四、 特种二极管
## 1. 稳压二极管



稳定电压：Vz (类似于0.7导通电压)
动态电阻：rz (越小，曲线越陡，稳压特性越好)
最小稳定电流：Iz(min)

简单的稳压电路：适用于电流较小，负载较轻的场合（并联）

## 2. 发光二极管

## 3. 光电二极管

## 4. 变容二极管

## 5. 肖特基二极管

导通电压：0.4左右
开关速度非常快，适合高频应用
