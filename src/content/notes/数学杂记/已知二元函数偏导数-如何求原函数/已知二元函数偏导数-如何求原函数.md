当已知一个二元函数的两个偏导数时，可以通过积分来求得原函数。这在求解全微分方程中是一个核心步骤。以下是详细的求解方法和步骤。

### 01. 求解前提：检验是否存在原函数
首先，并非任意两个函数 P(x, y) 和 Q(x, y) 都能成为某个二元函数 f(x, y) 的偏导数。必须满足一个前提条件，即混合偏导数相等。

假设已知：
*   ∂f/∂x = P(x, y)
*   ∂f/∂y = Q(x, y)

那么，必须满足以下条件，f(x, y) 才存在：
**∂P/∂y = ∂Q/∂x**

如果这个条件不成立，则不存在一个原函数 f(x, y) 同时满足给定的两个偏导数。

### 02. 求解方法
如果满足上述条件，主要有以下几种方法来求解原函数 f(x, y)。

---

#### **方法一：偏积分法（最常用）**
这是最直接和系统的方法，步骤如下：

1.  **对其中一个偏导数进行积分**：
    选择一个偏导数，例如 ∂f/∂x = P(x, y)，然后将其对 x 进行不定积分。在积分过程中，将变量 y 视为常数。
    > f(x, y) = ∫P(x, y)dx + g(y)

    **注意**：这里的积分常数不是一个简单的常数 C，而是一个关于 y 的函数 g(y)，因为任何只关于 y 的函数对 x 求偏导数都为零。

2.  **求出 g(y)**：
    *   将上一步得到的结果对 y 求偏导数：
        > ∂f/∂y = ∂/∂y [∫P(x, y)dx] + g'(y)
    *   将这个结果与已知的另一个偏导数 ∂f/∂y = Q(x, y) 相等：
        > Q(x, y) = ∂/∂y [∫P(x, y)dx] + g'(y)
    *   解出 g'(y)：
        > g'(y) = Q(x, y) - ∂/∂y [∫P(x, y)dx]
    如果前提条件 ∂P/∂y = ∂Q/∂x 满足，那么上式右边的 x 项将相互抵消，最终只剩下关于 y 的表达式。

3.  **积分得到 g(y)**：
    对 g'(y) 进行积分，得到 g(y)：
    > g(y) = ∫g'(y)dy + C

    这里的 C 是一个真正的常数。

4.  **写出最终结果**：
    将求出的 g(y) 代回到第1步的表达式中，即可得到完整的原函数：
    > f(x, y) = ∫P(x, y)dx + g(y) = ∫P(x, y)dx + ∫g'(y)dy + C

**示例：**
已知 ∂f/∂x = 2xy + 3 和 ∂f/∂y = x² - 4y，求 f(x, y)。

*   **第一步：检验条件**
    *   P(x, y) = 2xy + 3, Q(x, y) = x² - 4y
    *   ∂P/∂y = 2x
    *   ∂Q/∂x = 2x
    *   因为 ∂P/∂y = ∂Q/∂x，所以原函数存在。

*   **第二步：对 x 偏积分**
    *   f(x, y) = ∫(2xy + 3)dx = x²y + 3x + g(y)

*   **第三步：对 y 求导并比较**
    *   对上式求 y 的偏导数： ∂f/∂y = x² + g'(y)
    *   令其等于已知的 Q(x, y)： x² + g'(y) = x² - 4y
    *   解得：g'(y) = -4y

*   **第四步：积分求 g(y)**
    *   g(y) = ∫(-4y)dy = -2y² + C

*   **第五步：得出原函数**
    *   将 g(y) 代回，得到最终结果：
        > f(x, y) = x²y + 3x - 2y² + C

---

#### **方法二：凑微分法**
此方法适用于能够观察出全微分形式的方程。其核心思想是将表达式 P(x, y)dx + Q(x, y)dy 凑成若干个函数的全微分之和。

例如，已知 df = (2xy + 3)dx + (x² - 4y)dy，求 f(x, y)。
```
df = 2xydx + 3dx + x²dy - 4ydy
df = (2xydx + x²dy) + 3dx - 4ydy
```
观察到 2xydx + x²dy = d(x²y)，因此：
```
df = d(x²y) + d(3x) - d(2y²)
df = d(x²y + 3x - 2y²)
```
两边积分得到：
> f(x, y) = x²y + 3x - 2y² + C

---

#### **方法三：线积分法**
原函数 f(x, y) 也可以通过路径无关的线积分来求解。选择一个简单的路径，例如从一个方便的起点（如 (0, 0)）到点 (x, y) 的折线路径。
> f(x, y) = ∫(0,0)→(x,y) P(x, y)dx + Q(x, y)dy

选择路径 (0,0) → (x,0) → (x,y)：
*   从 (0,0) 到 (x,0)：y=0, dy=0
*   从 (x,0) 到 (x,y)：x 为常数, dx=0

> f(x, y) = ∫₀ˣ P(t, 0)dt + ∫₀ʸ Q(x, u)du + C

使用上面的例子：
```
f(x, y) = ∫₀ˣ (2t*0 + 3)dt + ∫₀ʸ (x² - 4u)du + C
f(x, y) = ∫₀ˣ 3dt + [x²u - 2u²]₀ʸ + C
f(x, y) = [3t]₀ˣ + (x²y - 2y²) + C
f(x, y) = 3x + x²y - 2y² + C
```

### 总结
在已知二元函数的偏导数时，求原函数的关键在于积分。偏积分法是最通用和基础的方法，而凑微分法和线积分法则为特定情况提供了更快捷的解题思路。无论使用哪种方法，都必须先检验混合偏导数是否相等的条件。
