# 1.引言

该系列笔记创建于Troy学习、使用的过程中，将随着学习的逐渐深入而不断完善，最终形成一个成体系的git入门教程和指南。
## 1.1 什么是Git

  Git是目前最强大、使用最广泛的分布式版本控制软件。简单来说，就是一个管理代码版本的工具。你是否遇到过这样的场景：当我们在入门编程时，往往会直接在原有代码上修改。当修改量变大时，我们会遇到修改后的代码效果更差，甚至根本跑不通，而原来的代码因为修改而丢失。这时候，我们也许会尝试将代码复制一遍，然后再副本上修改。如果效果补好，则重新回到原来的代码，尝试使用新的方式改进。而Git的作用，则是将这一手动、繁琐且易出错的过程系统化、自动化，从而极大地提高了开发效率和代码管理的可靠性。
